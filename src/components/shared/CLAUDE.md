# src/components/shared

Shared reusable form components for the OneLink Management Console.

## Files

- `FormTextField.tsx` - Standardized MUI text input with optional character counter
- `FormToggle.tsx` - Standardized MUI switch with label and helper text
- `FormSelect.tsx` - Standardized MUI select dropdown
- `SectionAccordion.tsx` - Collapsible section wrapper with optional required-field indicator
