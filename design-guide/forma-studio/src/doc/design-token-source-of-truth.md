# Design Token Source Of Truth

This design-guide project is a reference/playground.

Token authority is not defined here.

Use these files as canonical token sources:
- `design/design-system.pen`
- `design/tokens/design-tokens.generated.json`
- `design/tokens/design-tokens.generated.md`

Refresh generated docs from repository root:

```bash
pnpm tokens:sync
```

If this guide differs from generated token outputs, generated outputs win.
